# Android Emulator FRP service


## Project setup with yarn
```
npm i -g yarn
```

### Compiles and run the node server
```
yarn install
yarn run server
```

### Compiles and minifies for production (Only if needed)
```
yarn run build
```

## Fix Lint Issues

### This will fix lint issues (Run before creating PR)
```
yarn run lint
yarn run format
```
